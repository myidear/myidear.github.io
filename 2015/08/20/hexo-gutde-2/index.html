Hexo系列教程: (二)Hexo的Windows环境搭建

发表于 2015-08-20   |   分类于 Hexo ， 博客   |  

上一章讲解了对Hexo进行的简单介绍, 本章为您讲解Hexo在Windows环境下的搭建.

安装Git

去 Git 官网下载并安装即可.

安装Node.Js

去NodeJs 官网下载并安装即可.
注意: 尽量使用默认路径, 以保证环境变量的自动配置.

安装Hexo

使用npm命令即可直接安装 (可以直接在屏幕任意地方直接右键: Git Bash Here)


静静等待安装完毕即可.

创建Hexo目录

目录可以再随意地方创建 (注:建议使用全英文路径，避免不必要的问题)
EG: C:\Hexo (这是我的本地路径)
现在进入Hexo目录内在里面右键选择: Git Bash Here

Hexo 会自动生成建站所需要的文件

npm 会自动安装依赖环境
