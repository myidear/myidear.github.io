Hexo系列教程: (四)将独立域名与GitHub Pages的空间绑定

  • Blog
  • GitHub
  • Hexo
  • 博客
  • 教程
Hexo博客

恩，这里就不在讲述购买域名的方法了。国内购买域名建议去 “万网”，“美橙互联”等域名供应商。国外“GoDaddy”，其他的我也不太清楚了。

Github Page的设置

方法一：在Repository的根目录下面（项目的根目录），新建一个CNAME的文件（注：没有后缀）里面写入你要绑定的域名如：iread.io保存退出即可。

方法二：在你本地的Hexo项目根目录下找到source目录，在source根目录下创建CNAME文件注：没有后缀）里面写入你要绑定的域名如：iread.io保存退出即可。

DNS设置

在写搭建这个Blog的时候我也参考了很多文章，但是不知道为什么每篇文章都推荐DNSPod，这点我还是比较随意的。

这里使用 GoDaddy进行介绍。登录GoDaddy官网。

选择你要使用的域名，进行DNS管理

DNS ZONE FILE添加需要的记录。需要添加两条A记录，一条CNAME记录，如果去使用DNSPod还需要删除原来的NS记录，换上DNSPod提供的NS地址。

这两个地址是GitHub提供的。如果哪天Blog登陆不了了。可以去GitHub Page查看最新的IP即可。

HOST填写 www，POINTS TO填写你的 Git用户名.github.io。

搭建完成

至此，独立博客就算搭建完成，如有问题请在下留言。
